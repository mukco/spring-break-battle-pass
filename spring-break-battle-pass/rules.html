How the +5 minutes works

Each challenge can be redeemed for +5 minutes added to bedtime tonight. The minutes stack on the same day.

Cash option

Instead of bedtime minutes, a completed challenge can be redeemed for $5 cash. Each challenge can only be counted once.

  • 1 challenge = $5
  • 3 challenges = $15
  • 6 challenges = $30
  • 12 challenges = $60

Completion rules

  • The challenge must be finished.
  • The kid has to show what they made or did.
  • For bedtime rewards, finish before dinner to use the minutes that night.
  • Max bedtime bonus is 30 minutes for one night.
  • Unused bedtime minutes do not roll over to the next day.

Hosting at home with ngrok

Run a simple local server in the folder, then point ngrok to that port.

  • python3 -m http.server 8081
  • ngrok http 8081
